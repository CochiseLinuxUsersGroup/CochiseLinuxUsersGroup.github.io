CLUG Approved TRUSTED PERSONAL PACKAGE ARCHIVES (PPAs) List


01. Grub Customizer (submitted by Rex Bouwense)
-Grub Customizer is a graphical interface to configure the GRUB2/BURG settings and menuentries.
sudo add-apt-repository ppa:danielrichter2007/grub-customizer
sudo apt-get update
sudo apt-get install grub-customizer

02. Hand Brake (submitted by Tommy Worthey)
-HandBrake is an open-source, GPL-licensed, multiplatform, multithreaded video transcoder.
sudo add-apt-repository ppa:stebbins/handbrake-releases
sudo apt update
sudo apt install --install-recommends handbrake-gtk

03. Boot Repair (submitted by Rex Bouwense)
-Boot-Repair, simple tool to recover access to your Operating Systems. Easy-to-use (repair in 1 click!)
sudo add-apt-repository ppa:yannubuntu/boot-repair
sudo apt-get update
sudo apt-get install -y boot-repair && boot-repair
